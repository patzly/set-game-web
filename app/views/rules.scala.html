SET Spielregeln

Ziel des Spiels

Das Ziel des Spiels "Set" ist es, eine Gruppe von drei Karten zu finden, die zusammen ein Set bilden. Jede Karte zeigt Symbole mit vier Eigenschaften: Anzahl, Form, Farbe und Füllung.

Die Karten

Jede Karte hat eine Kombination aus den folgenden Eigenschaften:

  • Anzahl: 1, 2 oder 3 Symbole
  • Form: Oval, Welle oder Raute
  • Farbe: Rot, Grün oder Lila
  • Füllung: Voll, gestreift oder leer

Was ist ein Set?

Ein Set besteht aus drei Karten, bei denen jede Eigenschaft entweder komplett gleich oder komplett verschieden ist. Zum Beispiel:

  • Alle Karten haben dieselbe Anzahl Symbole, oder jede Karte hat eine andere Anzahl.
  • Alle Karten haben dieselbe Farbe, oder jede Karte hat eine andere Farbe.
  • Alle Karten haben dieselbe Form, oder jede Karte hat eine andere Form.
  • Alle Karten haben dieselbe Füllung, oder jede Karte hat eine andere Füllung.

Spielablauf

Zu Beginn werden 12 Karten offen auf den Tisch gelegt. Die Spieler versuchen so schnell wie möglich, ein Set zu finden. Sobald ein Spieler ein Set gefunden hat, nimmt er es vom Tisch, und drei neue Karten werden gezogen.

Spielende

Das Spiel endet, wenn keine Sets mehr gefunden werden können. Der Spieler mit den meisten Sets gewinnt das Spiel.
